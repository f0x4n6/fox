NAME
====

**fox** — The Swiss Army Knife for examining text files

SYNOPSIS
========

| **fox** **compare** \[_options_] \[_path_ ...]

DESCRIPTION
===========

Compare two files.

Alias
-----

diff, co

Options
-------

**-g, --git**

:   Use the unified git diff format.

EXAMPLES
========

fox compare server.log mirror.log
