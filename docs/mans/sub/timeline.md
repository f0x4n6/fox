NAME
====

**fox** — The Swiss Army Knife for examining text files

SYNOPSIS
========

| **fox** **timeline** \[_options_] \[_path_ ...]

DESCRIPTION
===========

Display super timeline.

Alias
-----

time, tl

Options
-------

**--cef**

:   Use the Common Event Format.

EXAMPLES
========

fox timeline --cef ./**/*.evtx
